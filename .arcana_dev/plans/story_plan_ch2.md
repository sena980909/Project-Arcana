# 📜 Story Architecture: Chapter 2 - 무너진 성채 (The Crumbling Citadel)

## 🎯 Theme Keywords
1. **배신(Betrayal)** - 믿었던 것에게 등을 찔리는 고통
2. **집착(Obsession)** - 사랑이 증오로 변하는 순간
3. **구원(Salvation)** - 구원받을 자격이 없는 자의 갈망

---

## 📊 Story Flowchart

```
[성채 입구: 폐허가 된 왕국의 문]
    │
    ▼
[외성: 첫 NPC '눈먼 기사' 조우]
    │ - 재의 상인 재등장 (펜던트 복선 회수)
    │
    ▼
[내성: 환경 스토리텔링]
    │ - 왕좌의 방 발견
    │ - 아이템 '깨진 왕관 조각' 획득
    │ - 벽화를 통한 과거 스토리 전달
    │
    ▼
[위기: 배신의 환각]
    │ - 하트 2개 이하 시 "네 친구도 널 버릴 거야"
    │ - 주인공의 과거 단편 암시
    │
    ▼
[탑 정상: 보스 조우]
    │
    ├─→ 발두르 - 몰락한 왕
    │   "사랑했다. 그래서 모두 죽였다."
    │
    ▼
[보스전]
    │
    ├─→ Phase 1: 왕의 위엄 (HP 100%~60%)
    │   - 느리지만 강력한 검격
    │
    ├─→ Phase 2: 광기의 폭군 (HP 60%~30%)
    │   - 빠르고 난폭한 공격
    │
    ├─→ Phase 3: 텅 빈 껍데기 (HP 30%~0%)
    │   - 절망적인 울부짖음, 자해 패턴
    │
    ▼
[결말: 발두르 소멸]
    │ - '무너진 성채의 아르카나' 획득
    │ - 주인공에게 의미심장한 말을 남김
    │
    ▼
[에필로그]
    - "넌 나와 같아. 언젠가 알게 될 거다."
    - Chapter 3 '침묵의 성당' 암시
```

## 👤 Characters

### 눈먼 기사 (The Blind Knight)
- **외형**: 녹슨 갑옷을 입은 기사. 투구 아래 눈이 있어야 할 곳에 흉터만 남아있다. 스스로 눈을 찔렀다.
- **비밀**: 발두르 왕의 가장 충성스러운 기사였으나, 왕이 미쳐가는 것을 막지 못했다. 그 죄책감에 "다시는 아무것도 보지 않겠다"며 자신의 눈을 찔렀다.
- **역할**: 보스 정보 제공, 성채 구조 힌트
- **결핍**: 충성을 바쳤던 왕이 괴물이 되어버린 현실을 받아들이지 못함

### 재의 상인 (복귀)
- **위치**: 성채 입구 근처 야영지
- **역할**: 펜던트 복선 회수, 상점 기능
- **추가 대사**: 펜던트를 보여주면 과거 이야기를 더 들을 수 있음
- **발전**: 이그드라의 죽음을 알고 복잡한 감정을 표현

### 보스: 발두르 (Baldur, The Fallen King)
- **외형**: 한때 위풍당당했을 왕. 지금은 반쯤 부패한 시체에 왕관이 살점에 박혀있다. 검은 눈물을 흘린다.
- **비밀**: 왕비를 너무 사랑한 나머지, 왕비가 병으로 죽자 "죽음을 정복하겠다"며 금지된 의식을 행했다. 그 대가로 왕국 전체가 저주받았다.
- **전투 패턴**:
  - Phase 1 (위엄): 느린 대검 휘두르기, 왕의 칙령(범위 공격)
  - Phase 2 (광기): 빠른 연속 베기, 분신 소환
  - Phase 3 (절망): 자해하며 폭발 공격, 울부짖음(전체 화면 흔들림)
- **죽음의 의미**: 사랑했던 것을 지키려다 모든 것을 파괴한 자의 해방

### 왕비의 환영 (선택적 조우)
- **조건**: 하트 1개일 때만 등장
- **외형**: 아름답지만 반투명한 여성. 주인공에게만 보인다.
- **대사**: "당신도 누군가를 사랑했나요? ...기억나지 않는 거죠?"
- **역할**: 주인공 과거 복선, 하트 1개 플레이어에게 추가 로어

## 🎭 Emotional Themes

### 발두르의 비극
> "사랑은 구원이 아니었다. 집착이었다."

발두르는 악인이 아니다. 그는 단지 사랑하는 사람을 잃는 것을 견딜 수 없었을 뿐이다. 그 '사랑'이 왕국 전체를 파멸시켰다.

플레이어는 발두르를 처치하면서 질문해야 한다:
- *"나도 누군가를 이렇게 사랑한 적이 있나?"*
- *"내가 기억하지 못하는 '그 사람'도 이렇게 잃은 건가?"*

### 눈먼 기사의 딜레마
> "충성은 맹목이었다. 진짜 맹목이 되어서야 깨달았다."

기사는 왕이 미쳐가는 것을 봤지만 막지 못했다. '충성'이라는 이름으로. 그래서 다시는 아무것도 보지 않기로 했다.

하지만 눈을 감는다고 죄가 사라지지 않는다.
