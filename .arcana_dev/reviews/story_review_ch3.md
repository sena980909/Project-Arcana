## 2. Major Story Revelation Check

### 🔴 Chapter 3 핵심 반전

**공개된 진실**: 주인공은 과거에 '사랑하는 사람'을 자신의 손으로 죽였다.

**검증 항목**:
- [x] 이전 챕터에서 암시가 있었는가? → Ch2 발두르 "넌 나와 같아"
- [x] 너무 갑작스럽지 않은가? → 고해실에서 단계적 공개
- [x] 플레이어가 소화할 시간이 있는가? → 에필로그에서 여운
- [x] 후속 챕터와 연결되는가? → Ch4 "피의 정원"에서 '그녀' 만남

**아직 숨겨진 것**:
- 죽인 사람의 정체 (얼굴이 보이지 않음)
- 왜 죽여야 했는지
- '???' 목소리의 정체

## 3. Gameplay Innovation: Silence Mechanic

### Phase 3 무음 연출 검증

| Aspect | Implementation | Risk | Mitigation |
|--------|----------------|------|------------|
| BGM 정지 | ✅ 완전 무음 | 플레이어 혼란 | 화면 이펙트로 "침묵 저주" 표시 |
| SFX 제거 | ✅ 발소리/공격음 무음 | 타이밍 파악 어려움 | 시각적 공격 전조 강화 |
| 대사 표시 | 자막만 표시 | 몰입도 유지 | 텍스트 연출 강화 |

**예상 플레이어 반응**:
1. "뭐지? 소리가 안 들려" - 혼란
2. "아, 보스 패턴이구나" - 이해
3. "이거 무섭다" - 의도된 공포
4. 보스 처치 후 소리 복귀 - 해방감

**설계 의도**: 플레이어에게 '침묵의 공포'를 체험시킴. 익숙한 청각 피드백 제거로 불안감 극대화.

## 4. NPC Dialogue Innovation: 필담 시스템

### 말 없는 수녀 대화 방식

```
기존 NPC: [Speaker]: "대사"
말 없는 수녀: [필담] 텍스트 (손글씨 폰트)
```

**특수 연출**:
- 손글씨 스타일 폰트 사용
- 쓰는 동안 애니메이션 (펜 움직임)
- 종이 넘기는 효과음

**서사적 의미**: 진실을 말하려다 입을 봉해진 자. '침묵' 테마의 인격화.

## 8. Localization Notes

### 번역 시 주의사항

1. **"마더 실렌시아"**: 가톨릭 수녀원장 호칭. 'Silence'와 유사한 이름
2. **필담 대사**: 손글씨 느낌 + 약간의 문법 불완전함 유지
3. **Phase 3 자막**: 모든 언어에서 "완전한 침묵" 효과 유지
4. **"왜?"**: 단 한 마디지만 가장 중요한 대사. 원망+슬픔+배신감 담긴 어조
5. **"신은 침묵하지 않았어. 우리가 듣지 않았을 뿐"**: 핵심 테마. 의미 정확히 전달
