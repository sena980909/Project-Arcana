## 3. The Twist: "사랑해서 죽였다"

### 설계 의도

**1~3챕터**: "나는 살인자다" → 플레이어에게 죄책감 유발
**4챕터 반전**: "죽이는 것이 사랑이었다" → 관점 전환

### 이 반전이 작동하는 이유

1. **리리아나가 먼저 부탁했다** - 피해자가 동의한 죽음
2. **세상을 구하기 위해** - 개인 vs 전체의 선택
3. **저주에 잠식되기 전에** - '그녀'로 죽게 해준 것
4. **리리아나 본인이 용서했다** - 원망이 아닌 감사

### 감정적 카타르시스

| 감정 흐름 | 순간 |
|----------|------|
| 죄책감 | 1~3챕터 내내 |
| 공포/충격 | 장미 미로 기억들 |
| 슬픔 | 보스전 |
| 용서받음 | 리리아나 "알고 있었어" |
| 희망 | "진짜 리리아나를 구할 수 있다" |

## 4. Boss Design: 감정 전투

### 리리아나의 환영 3페이즈

| Phase | HP | 테마 | 특징 |
|-------|-----|------|------|
| 1 | 100-65% | 사랑의 기억 | 아름다운 패턴, 장미 꽃잎 |
| 2 | 65-30% | 배신의 고통 | 가시/덩굴, 공격적 |
| 3 | 30-0% | 혼란/용서 | 불규칙, 울면서 공격 |

### Phase 3 특수성

**"미워해야 하는데... 사랑해야 하는데..."**
- 사랑과 원한 사이에서 갈등하는 리리아나
- 불규칙한 패턴 = 그녀의 혼란된 감정
- 플레이어가 '공격'이 아닌 '해방'시키는 느낌

## 9. Localization Notes

### 번역 시 주의사항

1. **"리리아나"**: 백합(Lily)에서 유래. 순수함/죽음의 이중성
2. **장미 색상별 의미**: 문화권별 장미 색상 의미 차이 주의
3. **"날 죽여줘"**: 부탁/간청의 뉘앙스 정확히. 명령이 아님
4. **"알고 있었어. 처음부터"**: 핵심 용서 대사. 따뜻하게 번역
5. **"진짜 나는 살아있다"**: 희망을 주는 톤
